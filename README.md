
# Stunning Website for Studio Baan

This is a code bundle for Stunning Website for Studio Baan. The original project is available at https://www.figma.com/design/Uf0JeFVWqscbLvAoiFeK10/Stunning-Website-for-Studio-Baan.

## Running the code

Run `pnpm i` to install dependencies.

Run `pnpm dev` to start the Next.js development server.

## Build for production

Run `pnpm build` to create a production build.

Run `pnpm start` to run the production server.
